ATMAN Invest Casting

데이터가 남긴 흐름을 읽고,
실행 가능한 시나리오로 바꿉니다

ATMAN Invest Casting은 단순히 강한 종목을 찾는 시스템이 아닙니다. 가격과 자금의 흐름, 섹터 안에서의 역할, 반복되는 행동 특성, 현재의 위험과 남은 공간을 함께 분석해 지금 이 종목이 어떤 역할을 맡을 수 있는지 판단하고, 조건이 맞을 때만 실행 가능한 시나리오로 연결합니다.

핵심은 예측이 아니라 대응입니다. 하나의 방향을 단정하는 대신, 목표에 도달하는 길·시간이 지연되는 길·판단이 무효화되는 길을 함께 준비합니다.

1. 하나의 종목이 실행 후보가 되기까지

여러 단계의 관측과 검증을 통과한 종목만 실제 실행 시나리오로 연결됩니다.

01

다각도 관측

같은 종목을 섹터·패턴·관계·자금 밀집의 네 가지 관점으로 살펴봅니다.

하나의 지표나 한 사람의 시선에 의존하지 않습니다.
02

종목별 행동 모델

반복되는 성공·실패·회복의 특징을 종목별로 축적해 살아 있는 행동 프로필을 만듭니다.

종목마다 다른 성격과 조건을 따로 기억합니다.
03

환경에 맞는 후보

시장과 섹터 환경을 먼저 확인해, 지금 검토할 가치가 있는 종목만 후보로 추립니다.

모든 종목을 매일 같은 기준으로 보지 않습니다.
04

현재 배역 발견

과거 모델과 오늘의 가격·자금 흐름을 비교해 지금 이 종목이 맡을 수 있는 역할을 찾습니다.

장르는 미리 정하지 않고 현재 데이터에서 발견합니다.
05

시나리오 계약

진입 조건, 기대 목표, 무효화 지점, 유효 시간을 하나의 실행 시나리오로 만듭니다.

좋아 보이는 종목보다 실행 가능한 종목을 선택합니다.
06

규칙 기반 실행

실시간 데이터가 약속된 조건을 충족할 때만 진입하고, 목표·손절·시간 만료를 규칙대로 처리합니다.

감정이 아니라 사전에 정한 조건으로 행동합니다.

2. ATMAN이 다르게 보는 네 가지 렌즈

같은 데이터를 네 개의 전문 관점으로 읽고, 마지막에 하나의 종목 행동 모델로 통합합니다.

S

섹터 렌즈

종목이 속한 섹터가 살아 있는지, 자금이 확산되는지, 그 안에서 선도주·후발주·조연 중 어떤 역할인지 봅니다.

P

패턴 렌즈

가격이 어떤 형태로 움직이고, 얼마나 오래 유지되며, 어떤 모습으로 실패하거나 회복하는지 관찰합니다.

T

텐서 렌즈

가격과 자금이 누가 먼저 움직이는지, 서로 정렬되는지, 충돌하거나 관계가 끊어지는지를 읽습니다.

C

밀집 렌즈

자금이 어디에 응축되고 과열되는지, 압력이 유지되는지 소진되는지, 해소에는 시간이 얼마나 필요한지 봅니다.

3. 강한 종목보다, 지금 탈 수 있는 파동을 찾습니다

ATMAN은 단순한 상승·하락보다 작은 변화가 더 큰 흐름으로 전달되는 과정을 중요하게 봅니다.

가격이 쉬어도 자금이 버티는가

짧은 급등 자체보다, 그 뒤의 숨고르기에서 상위 흐름의 자금이 새 가격대를 지지하는지를 확인합니다.

01
작은 파동의 변화기존 범위를 벗어난 의미 있는 움직임이 시작됐는가
02
새 공간의 유지되돌림이 나와도 새롭게 만든 가격 공간을 지키는가
03
자금의 인수단기 움직임이 줄어도 중기 자금 흐름이 그 공간을 받아주는가
04
큰 흐름과의 정렬중기 변화가 장기 흐름과 연결되거나 기존 구조를 바꾸기 시작하는가

현재 종목의 역할

모든 종목을 같은 시간과 같은 방식으로 매매하지 않습니다. 현재 데이터가 보여주는 역할에 맞춰 유효 시간과 대응 방식을 달리합니다.

단기 반응형짧은 파동만 살아 있고 상위 흐름의 확인이 부족한 상태
중기 전환형단기 변화가 중기 자금 흐름으로 넘어가기 시작한 상태
장기 추세형단·중·장 흐름이 연결되어 추세가 진행 중인 상태
역추세 회복형큰 흐름은 약하지만 단·중기에서 회복력이 나타나는 상태
역할은 입력값이 아니라 분석 결과입니다. 종목을 미리 특정 장르에 끼워 맞추지 않고, 현재 가격과 자금의 관계에서 오늘의 배역을 발견합니다.

4. 오디션은 ‘좋은 종목’을 고르는 과정이 아닙니다

좋은 종목이라도 이미 너무 올랐거나, 손실 위험에 비해 남은 공간이 작거나, 실행 기준이 불명확하면 통과하지 않습니다.

현재 유사성

과거의 강점이 지금도 살아 있는가

종목 모델에 축적된 행동 특성과 오늘의 가격·자금 흐름이 실제로 닮았는지 확인합니다.

남은 공간

이미 늦은 자리는 아닌가

추세가 좋아 보여도 목표까지 남은 공간이 작으면 실행 후보에서 제외합니다.

비대칭성

기대수익과 위험의 관계가 유리한가

목표·무효화·시간 불확실성을 함께 보며 단순한 숫자 하나로 합격시키지 않습니다.

실행 가능성

조건을 명확히 정할 수 있는가

진입·목표·청산·유효 시간을 구체적으로 만들 수 없는 종목은 분석이 좋아도 실행하지 않습니다.

5. 하나의 예측이 아니라 세 가지 대응 경로

A

목표 도달

정해진 유효 시간 안에 기대한 가격 구조에 먼저 도달하면 목표 경로가 완성됩니다.

B

시간 만료

목표도 무효화도 발생하지 않은 채 시간이 끝나면 계약을 연장하지 않고 재검토하거나 종료합니다.

C

판단 무효화

기대 근거가 깨지는 가격·자금 조건이 먼저 발생하면 손실을 인정하고 계약을 종료합니다.

AI는 관계와 의미를 해석하고, 실행 엔진은 정해진 조건을 감정 없이 확인합니다.
